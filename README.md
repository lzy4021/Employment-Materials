# Employment Materials
 已上传了个人关于非技术面问题的回答总结.
 首先强调最重要的一点,坚持投简历坚持面试,由于我们本科双非,一定要大量投,多参加宣讲会.
另外,很多网申和面试会有一些开放性非技术问题, 但是我在网上却找不到计算机专业学生在这方面的回答,所以我希望自己的答案能填补一下这个空白.
